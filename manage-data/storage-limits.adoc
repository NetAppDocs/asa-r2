= ASA r2 storage limits
:icons: font
:imagesdir: ../media/

[.lead]
All ASA r2 configurations support the FC, iSCSI, NVMe/FC and NVMe/TCP protocols. The maximum nodes per cluster for all ASA r2 configurations is two.

= What's new in ONTAP 9.18.1 for ASA r2 systems
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
Learn about the new capabilities available in ONTAP 9.18.1 for ASA r2 systems.

== SAN data migration

[cols="2,4" options="header"]
|===
| Update
| Description

| Support for SVM migration
| You can non-disruptively migrate a storage virtual machine (SVM) from an ASA cluster to an ASA r2 cluster.
|===


== Data protection
[cols="2,4" options="header"]
|===
| Update
| Description

| SnapMirror active sync
| SnapMirror active sync is supported on 4-node clusters.
|===



== Storage efficiency
[cols="2,4" options="header"]
|===
| Update
| Description

| Support for modification of snapshot reserve and automatic snapshot deletion
| You can modify snapshot reserve and enable automatic snapshot deletion to limit the among of space used for snapshots in your ASA r2 storage units.
|===

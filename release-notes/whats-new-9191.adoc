= What's new in ONTAP 9.19.1 for ASA r2 systems
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
Learn about the new capabilities available in ONTAP 9.19.1 for ASA r2 systems.

== Security
[cols="2,4" options="header"]
|===
| Update
| Description

| link:../secure-data/enable-anti-ransomware-protection.html[Support for ARP/AI with SnapMirror active sync relationships]
| Autonomous Ransomware Protection with Artificial Intelligence (ARP/AI) supports primary read-write storage units in SnapMirror active sync relationships on ASA r2 systems. ARP/AI runs on the active primary storage units, creates primary-only ARP snapshots, and provides machine-learning-based ransomware detection for those storage units. 

| link:../learn-more/additional-supported-ontap-features.html[Support for TLS hardware offload]
| You can offload Transport Layer Security (TLS) encryption and decryption for your entire cluster to supported Ethernet cards.  This reduces CPU overhead and improves performance for systems using TLS.

|===

== Storage resource management
[cols="2,4" options="header"]
|===
| Update
| Description

| link:../learn-more/additional-supported-ontap-features.html[Support for modifying default settings that control automatic QoS throughput ceiling increases]
| You can modify the default settings that control how QoS policy groups automatically increase the QoS throughput ceiling in response to short-term needs for higher performance. These settings cannot be modified in ONTAP 9.18.1 and earlier.
|===

== Upgrade
[cols="2,4" options="header"]
|===
| Update
| Description

| link:../administer/upgrade-ontap.html[Enhanced reporting for automated pre-upgrade checks]
| When executing automated pre-upgrade checks, errors and warning are displayed separately to help you easily identify issues that must be resolved before you upgrade.
|===
